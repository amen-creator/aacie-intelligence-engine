# 🚀 الدليل الشامل لنشر مشروع AACIE V6 للعالم (Deployment Guide)

المشروع الآن أصبح **Enterprise-Ready** وجاهزاً للعمل على بيئة إنتاجية حقيقية للجمهور. 
لقد قمت بإعداد بنية **Docker** متكاملة تشمل واجهة Next.js، وخادم FastAPI، وقاعدة بيانات PostgreSQL.

لديك طريقتان لنشر المشروع ليراه الجميع:

---

## 🟢 الطريقة الأولى: النشر الموحد عبر VPS (الأسرع والأسهل)
هذه الطريقة مثالية إذا كنت تمتلك خادماً سحابياً (مثل DigitalOcean, AWS EC2, أو Railway).

1. قم برفع المجلد الحالي (مشروعك) بالكامل إلى الخادم الخاص بك عبر GitHub.
2. تأكد من تثبيت `Docker` و `Docker Compose` على الخادم.
3. داخل مجلد المشروع، قم بتشغيل الأمر السحري التالي الذي سيقوم ببناء كل شيء وربطه ببعضه:
   ```bash
   docker-compose up -d --build
   ```
4. **ماذا سيحدث؟**
   - سيعمل خادم قاعدة بيانات **PostgreSQL** محلياً على المنفذ `5432`.
   - سيعمل خادم الذكاء **FastAPI** على المنفذ `8000`.
   - ستعمل لوحة تحكم **Next.js** على المنفذ `3000`.
   - سيرتبط كل شيء تلقائياً في شبكة داخلية آمنة. يمكنك حينها ربط نطاقك (Domain) به!

---

## 🔵 الطريقة الثانية: النشر المجاني عبر المنصات السحابية (Vercel + Render + Supabase)

إذا كنت ترغب باستضافة المشروع **مجاناً** ليراه العالم فوراً:

### 1️⃣ قاعدة البيانات (Supabase - مجاني)
- اذهب إلى [Supabase.com](https://supabase.com) واصنع مشروعاً جديداً.
- انسخ رابط الـ Database URL الخاص بك.

### 2️⃣ محرك الذكاء (FastAPI -> Render - مجاني)
- انقل ملفات المجلد `backend/` إلى مستودع GitHub مستقل.
- اذهب إلى [Render.com](https://render.com) وأنشئ **Web Service** جديدة.
- اربط المستودع الخاص بك، وحدد بيئة `Python`.
- في الـ Environment Variables، أضف المتغيرات التالية:
  - `DATABASE_URL` = (رابط Supabase الذي حصلت عليه).
  - `GROQ_API_KEY`
  - `JINA_API_KEY`
  - `SCRAPER_API_KEY`

### 3️⃣ واجهة التحكم (Next.js -> Vercel - مجاني)
- انقل ملفات المجلد `frontend/` إلى مستودع GitHub مستقل.
- اذهب إلى [Vercel.com](https://vercel.com) واضغط **Add New Project**.
- اربط المستودع، وسيقوم Vercel تلقائياً باكتشاف أنه `Next.js`.
- في الـ Environment Variables، أضف:
  - `NEXT_PUBLIC_API_URL` = (رابط خادم Render الذي حصلت عليه من الخطوة السابقة).

🥳 **مبروك!** بمجرد انتهاء الرفع، سيكون لديك رابط عالمي يمكن لأي شخص في العالم الدخول إليه ورؤية لوحة الرادار ثلاثية الأبعاد الخاصة بـ AACIE!

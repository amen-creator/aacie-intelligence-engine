دليل النشر المخصص باستخدام Vercel و Supabase

1 تجهيز قاعدة البيانات (Supabase)

سنستخدم Supabase كقاعدة بيانات PostgreSQL قوية ومجانية لتخزين الأخبار وتحليلات الذكاء الاصطناعي:

  1. اذهب إلى Supabase واصنع حساباً مجانياً واضغط New Project.
  2. ضع اسماً للمشروع (مثل: aacie-db) وأنشئ كلمة مرور قوية (احفظها جيداً!).
  3. انتظر دقيقة حتى تبدأ القاعدة.
  4. من القائمة الجانبية، اذهب إلى Project Settings ➜ Database.
  5. في قسم Connection string، اختر URI وانسخ الرابط الذي سيبدو هكذا:
    postgresql://postgres:[YOUR-PASSWORD]@db.xxxx.supabase.co:5432/postgres

2 استضافة محرك الذكاء FastAPI (Render)

ملاحظة هامة: لا يمكن رفع الـ Backend على Vercel لأن وكلاء LangGraph يحتاجون أحياناً 20-40 ثانية لتحليل الأخبار، وVercel يقطع الاتصال المجاني بعد 10 ثواني (Timeout). لذلك سنرفع الـ Backend مجاناً على Render.com.
  1. قم برفع مجلد backend/ فقط إلى مستودع جديد على GitHub.
  2. اذهب إلى Render.com واضغط New Web Service واربط مستودعك.
  3. اختر البيئة: Python 3
  4. في خانة البناء (Build Command) اكتب: pip install -r requirements.txt
  5. في خانة التشغيل (Start Command) اكتب: uvicorn main:app --host 0.0.0.0 --port 10000
  6. في قسم Environment Variables، انسخ المفاتيح التالية:
    • DATABASE_URL = (رابط Supabase الذي حصلت عليه في الخطوة الأولى).
    • GROQ_API_KEY = مفتاح Llama-3 الخاص بك.
    • JINA_API_KEY = مفتاح القارئ.
    • SCRAPER_API_KEY = مفتاح جلب الأخبار.
  7. اضغط Deploy. انسخ رابط الـ API الجديد الخاص بك من Render (مثل: https://aacie-api.onrender.com).

3 استضافة واجهة التحكم (Vercel)

هنا سنرفع الرادار البصري ثلاثي الأبعاد وواجهة Next.js الساحرة:

  1. قم برفع مجلد frontend/ فقط إلى مستودع آخر على GitHub.
  2. اذهب إلى Vercel.com واضغط Add New ➜ Project.
  3. قم باستيراد مستودع الـ Frontend الخاص بك. سيكتشف Vercel تلقائياً أنه مشروع Next.js.
  4. قبل الضغط على Deploy، افتح قسم Environment Variables وأضف المتغير التالي:
    NEXT_PUBLIC_API_URL = (الرابط الذي حصلت عليه من موقع Render في الخطوة الثانية).
  5. اضغط Deploy! 🚀

بمجرد انتهاء Vercel، ستحصل على رابط عالمي يمكن لأي شخص مشاركته والدخول منه لرؤية النظام يعمل بالكامل متصلاً بقاعدة بيانات Supabase!
